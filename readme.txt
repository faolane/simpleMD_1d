*********************************************
                simpleMD_1d
*********************************************
             F. Brieuc - 2017
*********************************************

simpleMD_1d is a simple Molecular Dynamics (MD) 
code for one dimensional systems focusing on 
thermostats and methods to take into account 
nuclear quantum effects.

The code is still in beta for now, most 
features have not been fully tested yet!
